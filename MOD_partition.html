First things first

  1. Crie um diretório de trabalho na raiz do seu computador. Evite muitos subdiretórios;
  2. Por favor, instale o programa IQ-TREE command-line.
  3. Para iniciar o programa, inicialmente, clique no menu Start (no canto inferior esquerdo da janela do Windows) e digite cmd. Após isso, pressione Enter.
  4. Por favor, vá até a pasta do IQ-TREE que você acabou de descomprimir, digitando na janela aberta de comando cd Downloads\iqtree-1.5.0-Windows (assumindo-se que o programa está na pasta Downloads e que a versão do programa é 1.5.0).

Achando o melhor esquema de partição:
  1. Abrir uma janela de comando e digitar bin\iqtree -s example.phy -p example.nex -m MF+MERGE;

    2. Podemos ainda achar melhores esquemas, seguido por uma busca e uma reamostragem com substituição ultrafast bootstrap:

  2. bin\iqtree -s example.phy -p example.nex -m MFP+MERGE -B 1000;

Escolha de modelos para cada partição.
  1. Abrir uma janela de comando e digitar bin\iqtree -s example.phy -m MFP;
